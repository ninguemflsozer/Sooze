Lei geral de Proteção de Dados (LGPD)

A Lei Geral de Proteção de Dados (LGPD) é a legislação brasileira que regula as atividades de tratamento de dados pessoais. Ela tem como como objetivo prtoteger os direitos fundamentais de liberdade e de privacidade, bem como o livre desenvolvimento da personalidade da pessoa natural.

Princípais pontos da LGPD:

  • Transparência: As organizações devem fornecer informções claras sobre a coleta, uso e armazenamento dos dados pessoais.
  • Consentimento: O Consentimento do titular deve ser obtido para o tratamento de dados, salvo exceções previstas na lei.
  • Finalidade: Os dados devem ser coletados para propósitos específicos, explícitos e legítimos.
  • Segurança: As organizações devem garantir a proteção dos dados pessoais contra acesso não autorizados e situações acidentais ou ilícitas.
  • Direitos dos Titulares: Os titulares dos dados têm o direito de acessar, corrigir, eliminar e portabilizar seus dados pessoais.

Para mais informações, visite o Site oficial da LGPD.
